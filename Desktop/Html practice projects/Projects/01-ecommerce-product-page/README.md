# 🛒 E-commerce Product Page — HTML Project

Short project that simulates a product page from an online store using only HTML.

---

## 🎯 Objective

Build a simple and well-structured product page to practice HTML fundamentals and semantic organization.

---

## 🧠 Problem

How to present product information clearly and logically for users using only HTML, without CSS or JavaScript.

---

## 🛠️ Solution

I structured the page into meaningful sections to improve readability and user experience:

* Product title as the main heading
* Product image to represent the item visually
* Description section explaining the product
* Benefits list highlighting key features
* Price section with strong visual importance
* Call-to-action button to simulate a purchase action

---

## 🧱 Project Structure

* **index.html** — Contains the complete page structure in HTML

---

## 🎓 What I Practiced

* Basic HTML document structure
* Semantic tags and content hierarchy
* Information organization
* User-focused layout thinking

---

## 🚀 Result

A clean and organized product page that simulates a real e-commerce layout.

---

## 🔮 Possible Improvements

* Add CSS styling
* Make the layout responsive
* Add JavaScript interactions
* Include product variations (size, color)

---

## ▶️ How to Run

1. Download the project folder
2. Open the **index.html** file in your browser
