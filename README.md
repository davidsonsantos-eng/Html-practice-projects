# 💻 HTML Practice Projects

Repository dedicated to practicing HTML through progressive, hands-on projects.
The goal is to strengthen structure, semantics, code organization, and problem-solving skills while building a solid front-end portfolio.

---

## 🎯 Purpose

This repository was created to:

* Practice HTML in real project scenarios
* Develop logical and analytical thinking
* Improve code organization
* Document learning progress
* Build a professional portfolio

---

## 📁 Projects

**Main concepts practiced:**

* HTML document structure
* Semantic elements
* Content hierarchy
* User-focused layout

### 🍽️ 02 — Restaurant Menu

Digital menu with categorized food sections.

**Main concepts practiced:**
